# 小总结

对于子数组问题，涉及区间问题（不需要修改的），一般可以用前缀和进行优化。

对于子序列问题，常使用动态规划的方式进行解决。比如说，LIS， LCS，形成某种等差数列的子序列等等利用子序列的某个性质进行求解的问题。
